## git branch:

Команда git branch может быть использована для отображения, создания или удаления веток. Для отображения всех существующих веток в репозитории введите:

 

``` 
git branch 
```

Для создания ветки:

``` 
git branch <имя-ветки>
```
 
Для удаления ветки:

``` 
git branch –d <имя-ветки>
```
